ReactJS шаблон лабораторной работы. В проекте настроен vite и несколько плагинов для него. Для начала, нужно запустить `npm install` (или `yarn install`) в корне проекта - подтянутся все зависимости.

Команда `npm run dev` поднимает dev сервер с удобным hot-reload

Команда `npm run build` соберет проект в единый минифицированный html файл, который будет лежать в `./build/index.html` - его и нужно загружать в микроконтроллер.
